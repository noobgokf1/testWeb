Replaced Aura Module

Info: New easier way to add auras to the game

How It Works: Rename Or Duplicate Existing folder inside : ReplicatedStorage -> AurasFolder -> AurasM. Inside That Folder there are diffrent values. Just like Syntex v4 but without module. Now Open Properties and click on any value inside the folder. for example OneIn allows you to change rarity of aura by chaning value of OneIn Object in properties. If You dont want ceratin value for example Biome leave it blank

Disclaimer: You can still use Aura Module but remove everything inside AurasM. you can't use Aura Module (orginal way) and AurasM at the same time

Mutated Auras

Info: A new frame inside Collection for mutation / skins of auras

How to: to make mutated aura open AurasM and then aura you want to be mutated. next click Mutated Value and check properties. Change Value in properties to the name of the orginal aura

Module How to: You can also use module (old way) and add mutated variable after comma to the aura you want to be mutated. then after u wrote: mutated = "", put name of orginal aura between quotes
